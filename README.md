# .github

GitHubのIssuesが大幅に進化。これは期待。

・Issueの一覧をボードとテーブル表示に切替可能
・親子関係
・タスクからIssueに変換可能
・スプリントやチームの列追加や、並び替えやフィルター設定を保存可能
・カスタムフィールドの追加（優先度、期限とか）
